# Heli Install

Use the repository's `install.ps1` or `install.sh` to bootstrap `.heli-harness/` and pointer files. Stop if `.heli-harness/HARNESS.md` already exists; use the update workflow for an existing installation so local state is preserved. Verify the required harness files after installation.
